My Projects

What i do

I make both web and mobile apps. Let's start with web apps. I'm a fan of JAM-Stack, using google sheet API or firebase as the back-end and static hosting on Github. But for heavy-function usage, I usually approach with Node.js and MongoDB. My mobile apps are usually built using hybrid methods, which involve wrapping a website in an app with tools like Cordova. The most comfortable and efficient language for me to build is JavaScript.
